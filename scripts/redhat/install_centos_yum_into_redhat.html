於 RHEL 安裝 CenOS YUM Package Manager

發表於 2009 年 11 月 23 日 由 Tsung

之前幫忙設定 Red Hat Enterprise Linux(RHEL) 的系統, 好的狀況是有原始光碟, 可以直接 mount 進來裝, 差的狀況就是自己找 Source code 回來編譯.

上次遇到發現有人將 CenOS 的 YUM 弄到 RHEL 上, 確實可以正常裝進 RHEL, 但是沒有測試使用 YUM 安裝的 Package 狀況如何(如果機器在腳邊, 才敢這樣子測. XD)

此篇紀錄在 RHEL 安裝 CenOS 的 YUM 的過程, 至於使用 YUM 後續安裝不確定穩定度.(話說, 跟這樣子做比起來, 直接把 RHEL 換成 CenOS 應該會更安心點. XD)

註: RHEL 找不到可以免費抓 Package(rpm) 的地方.

提供此資源的 官方網站: yum - Trac (Yum Package Manager)

在 RHEL 安裝 YUM Package Manager 的 安裝步驟

下述步驟參考此篇: Red Hat Enterprise Linux AS release 4 (Nahant Update 5) 使用 YUM 自動更新

  1. wget http://yum.baseurl.org/download/2.0/yum-2.0.8-1.src.rpm
  2. rpmbuild --rebuild yum-2.0.8-1.src.rpm
  3. cd /usr/src/redhat/RPMS/noarch/
  4. rpm -ivh yum-2.0.8-1.noarch.rpm
  5. vim /etc/yum.conf # 將 baseurl 換成下述 (換成台灣 Server)

    baseurl= http://mirror.centos.org/centos/4/os/i386/
    baseurl= http://mirror.centos.org/centos/4/updates/i386/

  6. yum update # 這樣子就可以用囉~
