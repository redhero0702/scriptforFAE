Do Less, Achieve More! – 5個提升開發效率的工具

重複的動作減少做，複雜的動作簡單做

身為開發人員，總是會不斷地去思考，如何更快更好地解決問題。在日常的軟體開發工作中，我們常常需要重複地輸入某些字串和指令，或是開啟某些應用程式來協助我們完成工作，這些其實都是相當繁雜的動作。本篇文章介紹5個能協助你大幅提升開發效率的工具，部分工具雖然只能運行在 Mac 作業系統上，但其概念都是一樣的，你可以試著在你使用的作業系統或環境，尋找類似的工具或方法來幫助你簡化重複性的動作，提升整體的作業效率，同時減少錯誤的發生。

1.Alfred




Alfred 跟 Mac 內建的 Spotlight (Control + Space) 很類似，可以協助你開啓 Mac 上的任何應用程式或是檔案，但是 Alfred 更專注在如何開啓應用程式，以及與應用程式之間的整合，其獨特的 Workflow 功能，讓你可以設定快捷鍵的組合，或是連結某些動作來達成自動化的目的。例如透過 Alfred 你可以：

  • 搜尋帶有某個標籤的 Evernote 筆記，或是根據關鍵字來搜尋筆記的標題，並在 Evernote 中開啟。
  • 搜尋 StackOverflow 的關鍵字，並直接在瀏覽器中開啟。
  • 在 Sublime Text 中開啟某個檔案或資料夾。

以上這些動作完全可以透過 Alfred 來執行，來達到去滑鼠化( Mouseless )的目的，大幅提升工作效率。另外，Alfred 有完善的社群支援，很多使用者會分享他們製作的 Workflow，你可以直接下載安裝，或是針對自己的需求進行些許客製化的修改。

2.Dash




RTFM 是每個開發人員都應銘記在心的準則。程式出問題？忘記某個 API 該怎麼使用？ 官方文件絕對是最好的幫手。Dash 幫你收集了常見的程式語言或是 Framework 的文件，讓你可以快速查詢，甚至是在離線的時候，還能夠繼續使用，避免因為沒有網路，而無法查詢文件的困擾。除此之外，在 Dash 中搜尋文件時，也會同時列出 Google 和 StackOverflow 的搜尋結果，方便你快速參考。

3.aText




在日常的工作中，我們經常會重複地輸入特定字串，例如，英文住址，信用卡卡號，在命令列中開啓 Apache 伺服器等，然而大腦空間實在有限，很難容得下這些字串和指令，每次要用到總不免輸入錯誤或是要重新查詢。aText 可以協助你快速完成這樣的動作，例如透過設定，你可以把 git push origin master 縮減成 gpom，縮減了輸入的時間，同時也可避免輸入錯誤。常常忘記指令怎麼打嗎？ aText 絕對是你不可或缺的工具！

4.z




工程師時常需要和命令列為伍，在不同的資料夾之間切換乃是家常便飯。但是資料夾何其多，有時候常常要找出資料夾位置就花了不少時間了。而 z 就是為了要解決這樣麻煩的事。安裝了 z 之後，你需要先在常用的資料夾目錄之間切換，z 會記憶你常使用的資料夾，之後你只需要輸入 z 加上部分的關鍵字，就可以快速切換到該目錄，進行後續的動作。

5.emmet




emmet 是前端工程師不可或缺的開發利器，支援 Sublime Text, Eclipse, Notepad++ 等多種編輯器，讓你可以快速產出 HTML 和 CSS。emmet 的概念其實跟 aText 很類似，都是透過縮寫來自動生成內容，差別在於 aText 需要自己定義縮寫，算是比較一般性的工具，而 emmet 則是專注在 Web 開發，將縮寫和自動化的動作幫你定義好了。還在一鍵一鍵慢慢寫網頁嗎？ emmet 讓你體驗前所未有的飛快速度！

你的工具箱裡有什麼利器？

以上介紹的工具概念都是很類似的，那就是“重複的動作減少做，複雜的動作簡單做”。想一想你的日常開發流程，是否有可以改善的的地方呢？如果有的話，不妨透過這些小工具來改善，並提升工作效率，成為一個更好的開發人員。當然每個開發人員一定都有屬於自己的工具和獨門祕技，若你有其他推薦的工具或是方法，也歡迎在底下留言跟大家分享。
