==update 2013-11-28T13:01:19+08:00==
已经做了一个周的iOS开发了，觉得iOS开发其乐无穷啊。
看到这么多人关注这个问题，还有点小激动呢（醒醒吧！又不是给你点赞！！）。其实我最初是想把重点放在“程序员”和“Mac”上，不过看见有很多知友从方方面面分享了Mac的使用经验，也受益匪浅。作为提问者，我还是想从程序员的角度补充分享一点经验。
  • 使用Homebrew — MacPorts driving you to drink? Try Homebrew!
这个绝对是程序员居家旅行必备的神器了，功能上跟apt-get差不多。

  • 注册一个美区账号
很多App在大陆是不发售的（比如下面这款应用），因此你需要去注册一个美区的Apple ID。
  • 使用Microsoft Remote Desktop
这个应用貌似不在大陆的商店中销售。这个应用特别适用这种情况：公司给配了一个台机，上面装了个Windows，然后你又想把台机的显示器分给Mac做双屏，偶尔用到Windows的时候还得把显示器还给台机，接来接去可麻烦（好吧，不要问我为什么不再去向公司申请一个显示器），这个时候这款神器就有用了，你可以远程桌面到你的台机！


==end==

使用rMBP快半年了，现在来回答自己提的问题（那些在Mac上装Win的就无视吧）。
声明：本人是程序员一枚，平时常用语言为Java、Python、Shell，会做一些Android开发以及脚本开发，喜欢折腾，见到优雅的可以称之为艺术品的应用就像见到了亲爹一样。
私以为，Mac的优点在于它的内在美：GUI和Unix命令行的完美融合，触控板和巧克力键盘带来的良好的用户体验。至于其他诸如外观、续航等就不说了。
我觉得程序员要优雅的使用rMBP可以充分做到以下几点：
  • 充分利用Mac的命令行（终端）
这是每个程序猿，不管做的是哪方面的技术都应该掌握的。一些基本的命令，会让你的工作效率提升很多。Mac自带的Shell是bash，建议换成zsh（Mac已经自带zsh，省去了安装的步骤，要做的只是切换一下，而且zsh兼容bash，使用起来无压力，其他优点不再赘述，自行Google），配合github上oh-my-zsh的插件，使用起来真心方便；此外，推荐使用自带终端的代替应用：iTerm2，也是个小神器了~网上这方面的教程很多。举个例子，比如要查看根目录一个隐藏文件夹的大小，用GUI怎么办？是不是要各种设置一番？用终端，一个命令就解决了：
iTerm2 + zsh，是我的标配了~
  • 使用Airdrop跟同事传文件
AirDrop是Mac自带的很棒的局域Wi-Fi无线传送技术，如果同事都用Mac，那么你们互传文件时尽量用Airdrop吧，UI很赞不说，速度也很快，不要再用QQ等IM的发送文件了。
  • 安装ohdarling/GoAgentX · GitHub GoAgentX
FQ应该是程序员必备的技能了吧~Mac下有一款很好用的图形化FQ工具，如题所示，这里是如何配置：如何在 Mac OS X 平台设置 GoAgent？
  • 主动去发掘一些应用
没事可以去逛逛V2EX等一些论坛，找到那些能提升你工作效率、生活品味的应用，多多把玩，才能让Mac物尽其用。像Sublime Text、Mou、OmniGraffle、EverNote、AppCleaner等等都不用说了吧~
  • last but not least：不要装Windows
这个只是个人的品味和建议了，我个人觉得装了Windows的Mac就特别不纯粹，可能是之前也一直再用Ubuntu，对Win没什么好感。实在不行了，安个虚拟机吧。

暂时想到这么多，一点个人的经验分享，还望海涵。

这样可以将窗口拖动到边缘自动调整大小。非常赞。

还是免费的！强烈推荐。

推荐几个小东西

1. Bartender 不到100块，将状态栏多余的图标都收起来，只能到官网用信用卡买。
2. JiTouch 不到50块，增强各种手势（Trackpad 和 Magic Mouse），只能到官网用信用卡买。
3. Caffeine 快速开启保持不会自动休眠，非常有用，免费，Mac Store。
4. iStat Pro 是一个 Dash Board 的插件，可以看到电脑的 CPU、内存、网络、电池的使用情况，貌似官网目前主推 App，不提供这个插件下载了，但还有很多网站留有下载链接。
5. Window Tidy 快速调节窗口大小，只要50块，App Store 有售。
6. Droplr 网盘，跟 OS X 结合非常无缝，截图自动生成链接到剪切板，App Store 有售。
7. Sip 屏幕取色，方便快捷，支持各种格式，6块钱，App Store 有售。
8. Screeny 录屏软件，好用，我用它录了很多软件操作视频，98块钱，App Store。

1. 替換光驅位，加裝SSD
參考教程：MacBook Pro 17" Unibody Optical Drive Replacement
吾的方式是把SSD裝到原硬盤位，能有6GBps的理論上限，機械硬盤放到光驅位，有3GBps理論上限。硬件換完后關鍵是要配置汝們的操作系統，這裏需要用到一些經驗，吾介紹一些，希望能有補充。
首先需要配置掛載方式，用sudo vifs進行配置（可以先設置好EDITOR環境變量）
UUID=CF6ABF48-BD75-3392-8582-683E36FED193 / hfs rw,noatime
吾用的是UUID指定硬盤，也可以用路徑，但是後面的文件系統和掛載選項請注意，一定要選擇noatime，否則每次訪問文件都會修改最後訪問時間的字段，造成大量讀寫。
接著吾建議安裝Trim Enabler: groths.org 可以優化SSD讀寫。然後請把用戶目錄移動到機械硬盤上，可以到System Preference - Users & Groups - Unlock - Right click user - Advanced Options裡面修改路徑，然後把用戶目錄rsync過去，重啟后生效。接下來汝就可以把大型軟體安裝到~/Applications下面了。
注：提到rsync，就不得不講一下Mac的特別版本，可以通過brew進行安裝，使用的時候用rsync -avxSEH可以保留HFS的特殊文件屬性（xattr）

2. 配置HTFS支持
不想買Paragon，就摸索了一下，吾已經總結了詳細步驟，見Mac OS X Mountain Lion NTFS Support Using NTFS-3G
大致翻譯一下主要步驟：安裝Homebrew，用brew裝ntfs-3g，替換Mac的NTFS掛載程式為自檢測腳本，done

（UPDATE）吾之前為了支持開源與免費一直在用NTFS-3G，但是到了後來實在忍受不鳥2MB/s的傳送速率。吾現在用的是Mac平台下讀寫最快的FUSE系統Tuxera NTFS for Mac，不過這是一個商業軟體，需要用31 USD買License。但是裝上這個后吾的讀寫速度有了大幅度提升。

3. 配置命令行終端
（吾的Terminal自製配色：XIR DARK by RixTox on deviantART）
吾認為Mac自帶的Terminal已經足夠優秀，沒有使用iTerm的絕對必要。以前用過fish shell，很喜歡它的簡潔，但是又想用回zsh的強大功能，後來發現zsh也有fishy主題:)
首先安裝oh-my-zsh: robbyrussell/oh-my-zsh · GitHub
然後編輯汝的.zshrc，根據自己的情況配置就好，吾推薦一些選項
COMPLETION_WAITING_DOTS="true"

plugins=(git brew osx sublime)

source $ZSH/oh-my-zsh.sh

# alias ls='ls --color'

alias l='ls'
alias ll='ls -alh'
alias la='ls -A'

吾這裏給出的幾個插件都是嚴重推薦的！omz的插件都很不錯，請汝自行選擇。
osx插件提供了一些很好用的命令，cdf能cd到最後一個處於激活狀態的finder目錄。sublime提供Sublime Text的相關命令，st [filename]能用ST編輯文件，stt能編輯當前目錄

10. iTerm2
之前吾不用它是因为那时它的字体优化做得非常渣，抗锯齿做得不忍直视。后来吾尝试了那时的Beta版，发现这个问题被解决后其实还蛮好用的说。现在吾就固定用iTerm了，吾把之前Terminal的那套配色照搬过来，用得还很爽。
在这里提一下关于iTerm在Alfred中配置workflow的问题，吾在网上找过很多写得不错的workflow，但是吾还想让吾的环境用起来跟tiling window manager一样舒爽，吾想让iTerm打开后固定在屏幕的右下角，占据1/4屏幕大小。所以吾把workflow跟slate稍微结合了一下，详细见代码。
on appIsRunning(appName)
  tell application "System Events" to (name of processes) contains appName
end appIsRunning

on alfred_script(q)
    if appIsRunning("iTerm") then
      tell application "iTerm"
        set myterm to make new terminal
        tell myterm
          launch session "Default Session"
        end tell
      end tell
    else
      tell application "iTerm" to activate
    end if
    tell application "System Events" to keystroke ">" using {shift down, option down}
end alfred_script
这里吾触发了吾的Slate热键Shift+Opt+>，详情请见吾上面的slate配置。这个workflow还可以应用在别的程式上面，吾将吾常用的开发软体都用这种方式固定在了屏幕的某个位置，真的有种在用i3的赶脚。
